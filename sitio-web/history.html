¿Cómo comenzó?

Kali linux primero comenzó como un proyecto llamado Whoopix, que significa WhiteHat Knoppix. Como lo dice el nombre, este estaba basado en el sistema operativo Knoppix. Whoppix tuvo lanzamientos desde la 2.0 hasta la 2.7.


Esto abrio paso para el siguiente proyecto, WHAX (O de forma más larga, WhiteHat Slax). El cambio de nombre fue porque el sistema operativo base fue cambiado de Knoppix a Slax. WHAX empezó en la versión 3 para continuidad con la version de Whoopix.


Había un sistema operativo similar siendo producido en ese tiempo, Auditor Security Collection (Comúnmente acortado a solo Auditor), otra vez usando Knoppix, y esfuerzos fueron combinados (con WHAX) para producir BackTrack. BackTrack estaba basado en el sistema operativo Slackware desde la versión 1 hasta la 3, pero luego cambió a ubuntu con la versión 4 hasta la 5.


Usando la experiencia ganada de todo esto, Kali Linux vino luego de BackTrack en el 2013. Kali empezó usando el sistema operativo Debian stable como la base hasta que se cambió a Debian testing cuando Kali se volvió un sistema operativo "rolling".
